# Student-Management-System
A console-based Student Management System developed using Java, JDBC, and MySQL to perform CRUD (Create, Read, Update, Delete) operations on student records. The project demonstrates the implementation of Object-Oriented Programming (OOP) concepts, database connectivity, input validation, and exception handling in Java.

# Features
➕ Add Student Records
📋 View All Students
✏️ Update Student Details
❌ Delete Student Records
✅ Input Validation and Error Handling
🔄 Exit Functionality
🗄️ MySQL Database Integration using JDBC
🧩 OOP-based Modular Structure

# Technologies Used
Java
JDBC
MySQL
OOP Concepts
Exception Handling
